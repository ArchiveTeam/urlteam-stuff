Welcome to the Urlteam website. The Urlteam is the ArchiveTeam subcomitee on URL shorteners. We believe that they pose a serious threat to the internet's integrity. If one of them dies, gets hacked or sells out, millions of links will stop working. Thus we preemptively release backups, because URL shorteners are too busy to make backups themselves.

Releases

Every 6 months we release a torrent of all backed up files. When a new torrent is released, you can simply delete the old torrent and download the new files in the same location. Your BitTorrent client will figure out which files have changed and will redownload them, even if you weren't finished yet on the previous torrent.

The latest torrent was released on February 8th, 2011: urlteam-data-2011-02-08.torrent

The next release is planned for August 2011.

Data format

All data files in the torrent are simple text files compressed using LZMA2/xz. The text file format is very simple: Each line contains one mapping in the following format: Shortcode, pipe (Ascii 0x7C), long URL, line feed (Ascii 0x10). The file is sorted by shortcodes using the following order:

  • Shorter shortcodes come before longer ones
  • Decimal digits
  • Lowercase letters
  • Uppercase letters
  • ASCII value

Depending on the URL shortener there might be multiple long URLs for one shortcode.

There are some tools for working with the files on GitHub.

Q&A

Can you do a backup shortener XY please?

Maybe. Some shorteners are very fast at banning scrapers which makes it impossible to do a backup in an efficient way. Contact us and we may look into it.

What about 301works.org? They help URL shorteners with backups.

And yet they haven't released a single file for download.
